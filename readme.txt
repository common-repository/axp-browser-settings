== Description ==

Mobile browser plug-in settings

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'AXP Browser Settings'
3. Activate BuddyPress from your Plugins page. (You will be greeted with a Welcome page.)

= From WordPress.org =

1. Download AXP Browser Settings.
2. Upload the 'AXP Browser Settings' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate BuddyPress from your Plugins page. (You will be greeted with a Welcome page.)
